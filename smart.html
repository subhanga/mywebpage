Smart Ligting System

Project Definition

Various engineering projects have constrained working with limited power supply. The device not being used can be put into sleep mode to conserve energy. This project implements smart streetlights.The light turns off and enters sleep mode after a minute if no object is detected within 10 cm.

The Goal

Read a sensor reading from Ultrasonic Sensor for the distance of how far an object is and if the space is more than 10 cm. Print the distance message concurrently and blink an LED while it is happening Go into deep sleep if the obstacle is more than 10 cm away from the sensor Demonstrate implementation of a low-power application with ESP-WROOM-32. Implementing FreeRTOS to make ESP32 perform different tasks concurrently without interrupting each other.

Components Used

  • ESP32 Devkit V1
  • Ultrasonic Sensor
  • Development Platform ESP-IDF
  • LED
