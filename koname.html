The Konane Game

The Goal

A Python Script that plays one side of a Konane game.

Introduction

Konane is a popular board game in Hawaii. An 8×8 board is populated with “X” and “O” symbols in a checkerboard pattern. The board rows are designated 0–7, and the columns are designated a–f. Two adjacent symbols are removed before the ga me starts. The driver program removes the same two starting symbols at every game.

Working

I used the minimax algorithm to implement the alpha-beta pruning. Minimax is an algorithm for determining the best move in a game. It finds the move that maximizes the gain for the player while minimizing their loss. But as the number of possible moves increases, this technique is not efficient. So then, I implemented the alpha-beta cut-off move. This new implementation ignores subtrees worse than any other one, thus increasing efficiency significantly.
