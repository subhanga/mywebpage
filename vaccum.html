Vaccum Pump Test Stand

Project Definition

Vacuum pumps were being tested inefficiently. The tester manually recorded the output data, and the value was calculated using an excel program. In addition, the Orifices opening was being done by approximating by the tester, thus being highly prone to error.

The Goal

The project goal was to develop a standalone program to calculate the pass-fail criteria for the vacuum pump. The calculation was performed using past data to increase accuracy, and all the information required for testing is displayed on the HMI screen such that a tester with no prior experience should be able to operate the program
